## 2024-05-18 - Semantic Modals and Escape Dismissal
**Learning:** Custom modals or overlays in the UI need semantic attributes (`role="dialog"`, `aria-modal="true"`), explicit `<button>` tags with `aria-label`s for close actions, and `Escape` key dismissal to adhere to the established accessibility pattern.
**Action:** Always add semantic roles, explicitly labeled close buttons, and `Escape` key handlers when creating or modifying modals in this application.
